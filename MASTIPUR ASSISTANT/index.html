MASTIPUR ASSISTANT

About

MASTIPUR ASSISTANT is a Python-based desktop assistant with a graphical user interface (GUI) built using Tkinter. It can respond to your queries, open applications, play music, fetch weather updates, and more.

Features
  • Text and voice-based interaction
  • Opens popular apps and websites (Google, YouTube, Steam, etc.)
  • Plays music from your computer or online
  • Fetches weather information
  • Simple and user-friendly interface
How to Run
  1. Install Python 3.x and required libraries (pip install -r requirements.txt).
  2. Download or clone this project to your computer.
  3. Run gui.py using Python:
    python gui.py
